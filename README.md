# Kvitter

A minimal Twitter-style feed built with React and Firebase — designed as a hands-on tutorial for Firebase Auth, Firestore, and optional Cloud Storage.

See **[SETUP.md](./SETUP.md)** for step-by-step instructions to clone this repo, connect your own Firebase project, and deploy to Firebase Hosting.

```bash
npm install
cp .env.example .env   # fill in your Firebase credentials
npm run dev
```
